# flutterdemo

一个基于 Flutter 构建的 RSS 阅读器示例，默认订阅「少数派」的 RSS 源，并提供 WebView 内嵌阅读体验。

## 功能概览

- 拉取 `https://sspai.com/feed` 并展示文章列表。
- 解析文章详情页的 `og:image`/`twitter:image` 作为缩略图，按正方形缩略图方式显示。
- 列表项提供标题、精简描述、作者信息与发布时间（`yyyy-MM-dd HH:mm`）。
- 点击文章后通过 `flutter_inappwebview` 打开内嵌网页，Android 端启用 Hybrid Composition 并移除页面切换动画，避免返回残影。
- 提供刷新功能，可手动重新抓取 RSS 与缩略图。

## 环境依赖

- Flutter 3.x+
- Dart SDK（随 Flutter 安装）
- 依赖包：`flutter_inappwebview`, `http`, `webfeed`

## 快速开始

```bash
flutter pub get
flutter run
```

默认会在首选设备（Android/iOS/macOS 等）运行。也可指定平台：

```bash
flutter run -d macos
flutter run -d android
```

## 平台注意事项

- **Android**：WebView 页面启用了 Hybrid Composition 并使用无动画路由，保证返回列表时的流畅度。
- **macOS**：`macos/Runner/*.entitlements` 已添加 `com.apple.security.network.client`，允许应用访问 RSS 与文章详情页。

## 调试建议

- 首次加载需要从文章详情页抓取缩略图，网络较慢时列表可能会短暂空白，可关注终端日志。
- 若站点对频繁请求有限制，建议按需增加缩略图缓存或限流策略。
